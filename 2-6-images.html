In this part of the second week, we are discussing about the image tag, which is more than a simple tag for inserting images!

I. SIZE

1. Why size matters?

Size of the image matters. As all the images must be downloaded in the browser, you must optimize the size and the quality of the image due to its importance and attention it gets.

On the other hand, if you the most of your audience paltform (i.e. using PC, mobile, tablet etc.), you can change the quality and size of the image to satisfy the most of you target audience.

NOTE: Each image requires an HTTP(Hypertext Transfer Protocol) request to be downloaded.

2. How to change the size of our images?

There are two main way exist:

  1. Change the original file size: You permanently changing the size of a local image file. So, what about linking to an image which the source is in the web?
  2. Use width/length attribute: We can either use pixels or percentage.

Example of using width/length attribute, for an image sourced on the internet

NOTE: Changing the size proportionally by percentage is of great importance! It is in fact proportional to 100% zoom of the browser and not the image itself. So, it plays an important role when you want your image to be looked perfectly in all the platforms!

II. OTHER FEATURES

1. How to add image in the title bar? (Web logo)

It is so interesting to have the logo of the website on the title bar! Here is how to do it. It must be written in the head section of the code before the title itself and not the body. You must use the <link> tag for this purpose:
<link rel="icon" type="image/png(ONLY PNG! YOU CANNOT USE jpeg OR gif)" href="" />

2. Alternative text (alt) attribute

The main aim of this attribute is to provide a textual alternative for a non-text content. For example, you can provide a text by which people who are not able to see the image for any reason (the outdated browser, not-convenient device and etc.), would be still able to have a sense of the image. We can list the puposes as below:

  • Provide info of the image for people using screen readers
  • Good alternative to be displayed instead of images
  • PROVIDING SEMANTIC MEANING OF AN IMAGE FOR THE SEARCH ENGINES. THIS COULD BE SO PRACTICAL.

Pay attention that the alternative text you are adding to an image be:

  • Accurate
  • Succinct (summarized)
  • Not being redundant
  • Not be a caption and explanation of the image!

NOTE: Remember that not all images must have had alternative text attribute. For example decorative and non-informative text can be without alternative text. To keep the structure of the perfect image tag, you can add alt attribute and put nothing inside the "" (leaving it null).
