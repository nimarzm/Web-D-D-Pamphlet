Attributes

These are simply orders, inside tags. Like the source attribute inside the image tag.
In the above example, "src" and "alt"(alternative text) are attributes of image tag.

Different attributes that can be used in image tag are:
  • src: Imprting the image.
  • alt: Infor for screen readers and broken links.
  • title: Text display on hover
  • class: Extra formatting, such as tumbnail
NOTE:The main format of any attributes are: "attribute="blah blah blah"".
